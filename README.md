# NVVSpeech Challenge

**Understanding and Generation of Speech with Nonverbal Vocalizations**
- Official website of the Challenge: [NVVSpeech Challenge](https://nvvspeech-challenge.github.io/)
  
- A special challenge session at **[ISCSLP 2026](https://iscslp.org/2026/)**.


## About

Human spoken communication extends beyond linguistic content — speakers frequently produce
nonverbal vocalizations (NVVs) such as laughter, sighs, crying, coughing, breathing, gasps, and
yawns. NVVSpeech promotes these vocalizations as a crucial yet under-explored component of spoken
language communication, with shared tasks in both **Mandarin Chinese** and **English**.

## Tracks

- **Track 1 — Nonverbal Vocalizations Understanding in Speech.** Given a speech recording, generate a transcript with inline NVV tags, identifying the spoken content and the category of each NVV.

- **Track 2 — Controllable Speech Generation with Nonverbal Vocalizations.** Given a transcript containing NVV tags, synthesize speech that naturally integrates the specified NVVs while preserving intelligibility, naturalness, and quality.

## NVV Label Set

breath, sniff, laugh, crying, cough, throat clearing, sneeze, sigh, gasp, snore, yawn, hum, moan,
hissing, lipsmack, burp.
